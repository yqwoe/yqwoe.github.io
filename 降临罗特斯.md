

## 降临罗特斯

| 螃蟹2  | 60%  |

| boss5  | 100% |
| boss6  | 100% |

# 1队输出

> 1. 尼雅BOSS： 快速大小三只小怪，削弱尼雅防御。**黄色读条出现，硬控技能打断。否则出现锤子哥，难度增加。**

| 序号 | 机制     | 说明                                                 |
| ---- | -------- | ---------------------------------------------------- |
| 1    | 吸氧     | 看自己蓝色读条过半，和队友一起打掉甲板两侧的触手吸氧 |
| 2    | 激光触手 | 激光瞬秒，触手拍打，在甲板两侧跳跃可躲过伤害。       |
| 3    | 小章鱼   | 随机捆住两名队友，需要迅速逐个解救队友。             |
| 4    | 幻影祭祀 | 迅速击杀，过程中注意脚下伤害。                       |
| 5    | 触手法阵 | 两名队员站在魔法阵，剩一队友打掉触手保护罩。重复两次 |
| 6    | 吸水     | 和普通章鱼一样疾闪躲避到章鱼两侧。                   |
| 7    | 十字爆炸 | 重点说，较普通伤害范围更多，需要提前做预判躲避。     |
| 8    | 水球     | 不需要障碍阻挡，正常躲避即可。                       |
